# Data Upload & Visualization Demo

This project is a lightweight proof-of-concept demonstrating:

- CSV/XLSX file upload
- Data preview in table format
- Chart generation (bar, line, pie)
- Clean and responsive UI

This demo is intentionally scoped to show the core flow.
Authentication, backend APIs, and editable Word export
will be implemented in the full application.

## Tech Stack
- React
- Chart.js
- PapaParse / XLSX
